# Simple Middle C

This is a simple web application that plays a continuous Middle C tone (261.63 Hz).

## Features
- **Pure Sine Wave:** Generates a smooth, gentle tone using the Web Audio API.
- **Background Playback:** Uses a special audio looping technique to keep playing even when you close the browser or lock your phone.
- **Visual Feedback:** The button changes color to indicate play/stop status.

## How to Use
1. Click the **Play** button to start the tone.
2. Click **Stop** to silence it.
